# Mexican-Toy-Store-Dashboard-using-Power-BI

A comprehensive Power BI dashboard designed to analyze sales, revenue, and product performance for a Mexican toy store chain. This project offers insights into store-level sales trends, product categories, profitability, and key performance indicators, supporting data-driven decision-making.

# Project Overview

This dashboard showcases a detailed analysis of toy sales across multiple stores in Mexico.
It helps identify:

Top-performing stores

Best-selling product categories

Profitability trends

Yearly and monthly sales performance

Stock distribution and supply insights

# Objective

The main objective of this report is to:

Monitor sales performance

Analyze store-level insights

Track category-wise revenue

Support strategic planning for inventory, marketing, and store expansion


Sales Target Data: Monthly Target, Store Target

Shipments/Stock: Stock on Hand, Ship Date, Vendor
